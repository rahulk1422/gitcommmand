Git & GitHub Commands

Your ultimate cheat sheet for version control

📦 Basic Git Commands

Command Description Source
git init Initializes a local Git repository. GitHub Guides
git clone [url] Creates a local copy of a remote repository. GitHub Guides
git status Shows the status of changes (untracked, modified, or staged). GitHub Guides
git add [file] Stages a specific file for the next commit. GitHub Guides
git add . Stages all new and modified files in the current directory. GitHub Guides
git commit -m "[message]" Records the staged snapshots permanently in version history. GitHub Guides
git pull Updates local branch with new commits from remote. GitHub Guides
git push Uploads local commits to the remote repository. GitHub Docs

🔀 Branching and Merging

Command Description Source
git branch Lists all local branches. GitHub Guides
git branch [name] Creates a new branch. GitHub Guides
git checkout [name] Switches to a branch. GitHub Guides
git checkout -b [name] Creates and switches to a new branch. GitHub Guides
git merge [branch] Merges a branch into current branch. GitHub Guides

💻 GitHub CLI Commands

Command Description Source
gh repo create Initializes a new GitHub repository. GitHub CLI
gh pr status Checks status of pull requests. GitHub CLI
gh pr create Creates a new pull request. GitHub CLI
gh issue list Lists open issues. GitHub CLI
